# Data Analytics Project

## Overview

This project demonstrates a complete end-to-end data analytics workflow. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL analysis using PostgreSQL, dashboard creation in Power BI, report generation, and presentation preparation using Gamma.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

---

## Dataset

* Imported the dataset using Python.
* Examined the dataset structure and data types.
* Identified missing values, duplicates, and inconsistencies.
* Prepared the data for further analysis.

## Project Steps

### 1. Data Loading

* Imported the dataset into Python.
* Performed an initial inspection of the data.

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distribution and summary statistics.
* Identified trends, patterns, and relationships.
* Visualized key insights using charts and graphs.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data formats.
* Prepared a clean dataset for analysis.

### 4. SQL Analysis (PostgreSQL)

* Imported the cleaned dataset into PostgreSQL.
* Wrote SQL queries to extract business insights.
* Used filtering, grouping, joins, aggregate functions, and sorting.

### 5. Power BI Dashboard

* Connected Power BI to the processed data.
* Built interactive dashboards with KPIs and visualizations.
* Added filters and slicers for better user interaction.

### 6. Report Creation

* Summarized the analysis and key findings.
* Documented insights and recommendations.

### 7. Presentation (Gamma)

* Created a professional presentation highlighting:

  * Project overview
  * Data analysis process
  * Dashboard screenshots
  * Key insights
  * Final conclusions

5. Open the Power BI (.pbix) file to explore the dashboard.
6. Review the project report and Gamma presentation for a complete summary.
